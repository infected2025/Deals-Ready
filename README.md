# Elvaro Deals Bot с парсером ManoMano

## Установка:
1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Добавьте переменную окружения TELEGRAM_TOKEN или создайте файл `.env`.

3. Запустите бота:
```bash
python bot.py
```

## Команды:
- `/start` — запуск
- `/deals` — показать предложения (поиск "bosch")
